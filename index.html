What is Script?

A script is a set of instructions that is executed by a computer or other device.

  • Server-side scripts run on the server and generate dynamic content.
  • Client-side scripts run on the client's web browser and interact with the user.

Examples of scripts include JavaScript, Python, and Ruby.
